Rich preferences, ultimate customization

Automato is a very flexible application. You can see lots of different configurable parameters on the Preferences page.

Profile switcher

There's a top-level control group that allows you to manipulate application profiles. Here you can create, remove, or select profiles. When a profile is selected, almost all the settings you see are stored under this profile. Reset to defaults resets almost all the settings but only for the current active profile.

Common settings tab

Work Phase time and Rest Phase time are the main settings to configure working and resting time intervals.
The Sound volume setting is applied to all sounds played by the app in all phases.
End of Work phase sound and End of Rest phase sound controls allow you to change melodies or even disable them by selecting the "--- Not selected ---" dropdown option.

Automation settings tab

Note: Focus and Overwork phases are used only when Automato is in the "Auto" mode!

Focus Phase time setting allows you to set a time interval during which the application expects you to make persistent activity. The Focus Phase precedes the Working Phase and the main purpose is to prevent accidental activation of the Working Phase. For example, if you accidentally press any key or move the mouse, then the Focus Phase starts, but if no activity is shown during the next 5 seconds, then the timer will be reset.

The Overwork Phase starts right after the Work Phase finishes. Automato immediately notifies you about the end of the Work Phase and waits for you to stop making any activity. The Inactivity time to consider you're away setting determines the timeframe you should not do any activity at all to activate the Rest Phase.

To make Automato fully autonomous, it's suggested to enable these settings as well: Start the app automatically on system startup and Start the timer automatically on application startup.

Detection settings tab

Note: All the settings in this tab are only applied when Automato is in the "Auto" mode!

There are settings for two similar scenarios:

  • When no activity has been detected during the Work Phase.

  • When a persistent activity has been detected during the Rest Phase.

In both cases, Automato notifies you about it. If you ignore the notification, Automato performs according to measures. For example, if you're still working during the Rest Phase, then Automato will turn on the Work Mode with 1 minute left. Another example: if you're not doing anything with the computer during the Work Phase, then Automato will show the popover with the notification. If you press any key or move the mouse, or touch the touchpad, then Automato will hide the warning. If you ignore the warning, the Focus Phase will be activated.
