App usage modes

There are three levels of automation to choose from.

Manual mode

Requires you to start and stop timers manually. This can be useful when you want to have full control over the application. Also, manual mode is useful when you learn something from video and do not actively interact with the computer (which makes Auto mode not so useful).

Semi-automatic

This mode is almost the same as Manual mode. You have two toggles here. The first one allows you to start the Work phase automatically (right after the Rest phase). The second one allows you to start the Rest phase automatically (right after the Work phase). If both toggles are enabled, the Work and Rest phases will run one after another in a cycle. All you need is to react when it's time to work or when it's time to rest.

Auto mode

This is a very advanced mode. The app tracks your activity (key presses, mouse movements, touchpad gestures) and automatically starts and stops the timers. It also can react when you're not active when you should be working and react when you're active when you should be resting. This mode has various settings to tune the application's behavior according to your preferences.

Note that the Auto mode requires the Accessibility permission in the macOS "Security & Privacy" system settings.
